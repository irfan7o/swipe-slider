# **App Name**: Card Swiper

## Core Features:

- Swipeable Card UI: Implements a horizontally swipeable card layout using Next.js.
- Card Stacking: Display three cards, where the center card is prominent and the side cards are partially visible in the background. Bring a card to the front when swiped.
- Gesture Handling: Handles user swipe gestures to transition between cards.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to provide a professional and calm feel, promoting user focus.
- Background color: Very light gray (#F5F5F5) for a clean, distraction-free interface.
- Accent color: Soft violet (#9575CD) as a complementary accent to hint at the user's possible next steps.
- Body and headline font: 'Inter' sans-serif for clear, modern readability throughout the application.
- The layout should prominently display the current card in the center, with the adjacent cards peeking from the sides. Ensure smooth transitions and intuitive swipe interactions.
- Subtle animation effects during card transitions to enhance user experience.
